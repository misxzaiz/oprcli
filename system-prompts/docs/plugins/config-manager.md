配置管理系统

## 使用方法

```
// 在对话中使用
@config-manager [参数]
```

## API

- `get`: (key) => server.configManager.get(key)
- `set`: async (key, value) => await server.configManager.set(key, value)
- `addTool`: async (config) => await server.configManager.addTool(config)
