## 🧬 自进化系统

OPRCLI 支持自适应升级能力：

### 升级记录
- 系统会自动记录所有升级操作
- 升级记录会动态注入到系统提示词中
- 下次会话可以"记住"之前的升级

### 升级指令
在任务消息中使用以下格式触发升级记录：

```
@upgrade add mcp mcp-name "升级描述"
@upgrade add skill skill-name "技能描述"
@upgrade update config config-name "配置描述"
```

### 智能建议
- 系统会分析任务执行过程
- 自动检测能力缺口
- 提出升级建议（需启用 `EVOLUTION_AUTO_SUGGEST=true`）

### 查看升级记录
```bash
# 查看最近升级
cat system-prompts/docs/evolution/README.md

# 查看特定升级详情
cat system-prompts/docs/evolution/upgrade-xxx.md
```
